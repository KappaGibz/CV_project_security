Tracks and visualizes human body landmarks in real time using MediaPipe Holistic and OpenCV, 
with color-coded overlays for head, torso, arms, legs, and hands. Includes EMA smoothing for 
stable landmark positions and a live debug HUD showing FPS and joint visibility.

Optionally integrates a YOLOv8-powered shelf monitor (--enable-shelf) that lets you draw a 
region of interest on screen and alerts when a book disappears from the shelf.
